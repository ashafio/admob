Flutter AdMob Demo App

A simple Flutter project for learning and testing Google AdMob ads (Banner, Interstitial, and Rewarded ads) using official test ad units, no AdMob account required.

This project is designed for beginners who want to understand how ads work in Flutter before moving to production.

🚀 Features
📢 Banner Ad (persistent bottom ad)
📺 Interstitial Ad (full-screen ad)
🎁 Rewarded Ad (earn coins by watching ads)
🎨 Clean Material 3 UI
🧪 Uses official Google test AdMob IDs (safe & free)
📱 Works on Android emulator & real devices
📦 Tech Stack
Flutter
Dart
google_mobile_ads plugin
Android (Kotlin backend default)

⚙️ Setup Instructions
1. Clone the repository
git clone https://github.com/ashafio/admob.git
cd flutter-admob-demo
2. Install dependencies
flutter pub get
3. Run the app
flutter run
🛠 Requirements
Flutter SDK (latest stable recommended)
Android Studio / VS Code
Android Emulator or physical device
Minimum Android SDK 23+
📌 Notes
❌ No AdMob account required (using test ads)
❌ No real ads are shown (safe for development)
